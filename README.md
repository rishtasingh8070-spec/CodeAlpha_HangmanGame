# CodeAlpha_HangmanGame
A Python Based game to guess a Random Pre-defined word.
🎮 Excited to share my first Python project — a Hangman Game!

I built a simple command-line Hangman game using Python that allows users to guess a hidden word letter by letter while tracking attempts and guessed letters.

Key features:
✅ Random word selection
✅ Input validation
✅ Guess tracking
✅ Win/Loss conditions
✅ Interactive gameplay

This project helped me strengthen my understanding of Python fundamentals such as loops, conditional statements, lists, functions, and the "random" module.

Every small project is a step toward becoming a better developer, and I’m continuing to learn and improve through hands-on practice.

Feedback and suggestions are always welcome!
